Актуальность

На данный момент в саратове нет аналогочных интернет-магазинов, имеющих возможность составить конкуренцию нашему, так как уступают в скорости, дизайне и качестве интерфейсов.

Цели и задачи

Целью дипломной работы является разработка интернет-магазина "Кофеёк", предназначенного для продажи разнообразной продукции, в том числе кофе, чая и принадлежностей для них.
Для решения цели нужно выполнить следующие задачи:

  • Исследовать специфику и особенности работы современных интернет-магазинов.
  • Спроектировать и создать прототип будущего интернет-магазина.
  • Разработать интернет-магазин, позволяющий пользователю быстро и в удобной форме приобрести необходимые ему товары.
  • Провести тестирование разработанного веб-приложения.
  • Рассчитать экономические показатели разработки.

Используемое ПО

В процессе разработки приложения были использованы следующие технологии:

  • HTML5;
  • CSS3;
  • JavaScript;
  • JavaScript UI library React;
  • JavaScript library Redux;
  • И некоторое количество иных вспомогательных технологий.
